# Vue 3 + Vite

This project is a learning journal for Vue.js, structured to help you understand and implement Vue 3 features using Vite as the build tool.

## Introduction
This template should help get you started developing with Vue 3 in Vite. The template uses Vue 3 `<script setup>` SFCs. Check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) to learn more.

Learn more about IDE Support for Vue in the [Vue Docs Scaling up Guide](https://vuejs.org/guide/scaling-up/tooling.html#ide-support).

## Getting Started
To get started with this project, clone the repository and install the dependencies:

```bash
# Clone the repository
git clone <repository-url>

# Navigate into the project directory
cd Day2LearningVueJs

# Install dependencies
npm install
```

## Project Structure
The project structure is as follows:

```
Day2LearningVueJs/
├── index.html
├── package.json
├── README.md
├── vite.config.js
├── public/
└── src/
    ├── App.vue
    ├── main.js
    ├── style.css
    └── components/
        ├── BaseCounter.vue
        ├── BaseFacts.vue
        ├── BaseList.vue
        └── BaseProducts.vue
```

## Components
- **BaseCounter.vue**: A simple counter component.
- **BaseFacts.vue**: Displays interesting facts.
- **BaseList.vue**: Renders a list of items.
- **BaseProducts.vue**: Showcases product information.
